# PoE Crafting Tool

> Intelligent crafting automation for Path of Exile.  
> Built for crafters who know what they want.

---

## Overview

If you've ever spent four hours alt spamming a +1 Amulet or trying to hit a 
3-notable Cluster, you know the pain. PoE Crafting Tool was built because dumb 
clickers don't understand mod weights or prefix/suffix logic. This one does.

---

## Features

### Crafting Modes
| Mode | Description |
|---|---|
| **Alt / Aug Spam** | Full prefix/suffix targeting with intelligent Augment support |
| **Regal Phase** | Magic → Rare transition with separate mod condition tables |
| **Chaos Spam** | Rare item rerolling with weight-based mod targeting |
| **Alch / Scour** | Alchemy + Scour cycle automation |
| **Essence Spam** | Essence tab integration, Normal and Rare base support |
| **Fusing & Sockets** | 6-socket and 6-link automation with automatic Jeweller recovery |

### Chain Crafting
- **Single Item** — craft one item at a time
- **Inventory Chain** — batch craft an entire inventory grid
- **Stash Chain** — craft directly inside a stash tab, including quad-tabs

### Mod Targeting
- Prefix / suffix type awareness on all conditions
- Numeric thresholds — e.g. "at least 80 life"
- Weight system — require one mod or a weighted combination
- Group mode — match any one set from a list of valid mod combinations
- Cluster Jewel CSV import with automatic prefix/suffix detection by position

### Quality of Life
- Micro-jitter for human-like mouse movement
- Stall detection with automatic currency reselection
- Backup currency stack rotation across up to 14 extra slots
- Quick Currency Set overlay — drag, resize, and apply all positions at once
- Mod picker with search, pool builder, and per-table targeting
- Named save/load profiles per crafting mode
- In-app tooltips on every feature

## Installation

1. Download the latest release from the [Releases](../../releases) page
2. Extract the ZIP to a folder of your choice
3. Run `PoECraftingTool.exe` as Administrator
4. Follow the in-app setup tooltips

---

## Setup (Quick Start)

1. Open Path of Exile in **Borderless Window** mode
2. Open your stash / currency tab in-game
3. In the bot, go to **Currency Position** and bind your currencies
4. Set your **Item Position**
5. Add your target mods and press **Start Crafting**

> Detailed guides and FAQ are available in our Discord server.
